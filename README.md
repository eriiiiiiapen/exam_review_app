# 社労士試験 論点管理アプリ

## 概要
社労士試験の学習において、各科目・論点ごとの理解度や復習状況を管理するためのWebアプリケーションです。

自身が労務事務として実務に携わる中で、
「制度や論点を“理解したつもり”のまま進んでしまう」
「どこが弱いのか可視化しづらい」
と感じた経験から、本アプリを作成しました。

## 作成背景
私はこれまで労務事務として、給与計算や社会保険・雇用保険手続きに携わってきました。
現在は社労士試験の学習を進めていますが、実務経験がある分、
「分かっているつもりになっている論点」があることに気づきました。

そこで、
- 論点単位で理解度を管理する
- 苦手な論点を可視化する
- 復習の優先度を判断できる
ことを目的として、本アプリを開発予定です。

## 主な機能
- ユーザー登録 / ログイン機能
- 科目・論点の登録
- 論点ごとの理解度管理（enum）
- 学習履歴の記録
- 理解度・未復習論点の可視化

## 使用技術
- Ruby 3.3
- Ruby on Rails 8.1.2
- Docker
- Git

## 補足
本アプリは学習支援を目的としたものであり、
制度の計算結果や法的判断を提供するものではありません。
